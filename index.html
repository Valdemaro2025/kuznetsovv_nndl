Deployment Notes

How to deploy on GitHub Pages (no server needed):

  1. Create a new public GitHub repository (e.g., titanic-tfjs).
  2. Add index.html and app.js to the repo root.
  3. Commit + push to main.
  4. Open repo Settings → Pages.
  5. Set Source to Deploy from a branch, choose main and / (root), then Save.
  6. After GitHub Pages builds, open the provided URL (it will look like https://<user>.github.io/<repo>/).

Tip: Keep the Kaggle CSVs local; you’ll load them via the file pickers at runtime.
